## 📦 本地开发

```bash
# 克隆仓库
git clone https://github.com/xfengyin/xfengyin.github.io.git
cd xfengyin.github.io

# 安装依赖
bundle install

# 本地预览
bundle exec jekyll serve

# 访问 http://localhost:4000
```

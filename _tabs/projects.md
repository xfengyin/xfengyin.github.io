## XingJu 星聚

<div class="project-card">
<h3>⭐ XingJu 星聚</h3>
<p><strong>跨平台聚合应用</strong> - 一站式音乐/视频/小说/漫画搜索平台</p>

<h4>技术栈</h4>
<ul>
<li><strong>后端</strong>: Rust + Tauri v2</li>
<li><strong>前端</strong>: React 18 + TypeScript</li>
<li><strong>样式</strong>: Tailwind CSS + 自定义赛博朋克主题</li>
<li><strong>状态</strong>: Zustand</li>
<li><strong>测试</strong>: Vitest + React Testing Library</li>
</ul>

<h4>核心功能</h4>
<ul>
<li>🎵 音乐聚合 - 网易云/QQ音乐/酷狗多源搜索</li>
<li>🎬 视频聚合 - 多平台视频资源</li>
<li>📚 小说聚合 - 网络小说搜索</li>
<li>📖 漫画聚合 - 漫画资源聚合</li>
</ul>

<h4>项目亮点</h4>
<ul>
<li>赛博朋克UI设计 - 霓虹风格 + 动态粒子效果</li>
<li>性能优化 - 图片懒加载、粒子自适应、代码分割</li>
<li>测试驱动 - 完整的单元测试覆盖</li>
<li>CI/CD - GitHub Actions自动构建多平台安装包</li>
</ul>

## Zen

<div class="project-card">
<h3>🧘 Zen</h3>
<p><strong>Python模块化执行框架</strong> - 支持依赖图与隔离执行</p>

<h4>技术栈</h4>
<ul>
<li><strong>语言</strong>: Python 3.10+</li>
<li><strong>依赖解析</strong>: 自定义依赖图算法</li>
<li><strong>隔离执行</strong>: 虚拟环境隔离</li>
<li><strong>LLM集成</strong>: OpenAI API</li>
</ul>

<h4>核心特性</h4>
<ul>
<li>🔮 意图解析 - 自然语言转执行计划</li>
<li>📊 依赖图 - 可视化模块依赖关系</li>
<li>🔒 隔离执行 - 沙箱环境确保安全性</li>
<li>🎯 精准执行 - 按需执行，避免冗余</li>
</ul>

<h4>应用场景</h4>
<ul>
<li>自动化工作流编排</li>
<li>AI Agent 执行引擎</li>
<li>复杂任务分解与执行</li>
</ul>

</p>
